## Tecnologias

- Git
- Sqlite3
- Python
  - Django
  - Django rest framework
  
 ## Como executar
 
 ```bash
 git clone https://github.com/guilhermelbo/api_movies
 
 cd api_movies
 
 pip install -r requirements.txt
 
 python manage.py migrate
 
 python manage.py runserver
 ```
